# Task Dashboard (React)

A simple and responsive task management dashboard built with React and deployed online using Vercel.

## Live Demo

https://task-dashboard-mdjeumo.vercel.app

## GitHub Repository

https://github.com/mdjeumo/task-dashboard

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Clear completed tasks
- Persistent storage using localStorage
- Automatic deployment with Vercel

## Technologies Used

- React (Hooks)
- JavaScript (ES6+)
- Vite
- HTML5 / CSS3
- Git & GitHub
- Vercel deployment

## What I Learned

Through this project I practiced:
